# trynmaps-backend
Backend for the Tryn Maps webapp

## Getting started

1. Fork this repo.
2. run `docker build -t trynmaps-api .`.
3. run `docker run -p 4000:4000 trynmaps-api:latest`.

See our getting started guide for contribution and deployment guidelines.
